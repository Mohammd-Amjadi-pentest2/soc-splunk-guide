🔄 What is Lateral Movement?

After gaining initial access, the attacker pivots to other systems to reach higher-value targets (Domain Controller, file servers, databases). This is done using legitimate admin protocols like RDP, SMB, WMI, or PsExec, making detection harder.

🔥 Common Techniques

RDP (T1021.001): GUI remote access, port 3389. EventID 4624 Logon_Type=10
SMB/PsExec (T1021.002): Installs temp service PSEXESVC, EventID 7045
WMI (T1021.003): Remote command execution, port 135, spawns wmiprvse.exe
Pass-the-Hash: NTLM auth with stolen hash, EventID 4624 Logon_Type=3 with NTLMv2

🚨
Critical Signal: Domain Controller accessed? Assume full domain compromise. Activate IR Plan immediately.
SPL · RDP Lateral Movement (Logon_Type=10) Windows Security · -24h
index=* sourcetype=WinEventLog:Security EventCode=4624 Logon_Type=10 earliest=-24h | stats count as rdp_logins, dc(ComputerName) as target_hosts, values(ComputerName) as host_list by Account_Name | where target_hosts > 3 | eval risk = case(target_hosts>10,"CRITICAL",target_hosts>5,"HIGH",1=1,"MEDIUM") | sort -target_hosts
ComponentExplanation
Logon_Type=10RemoteInteractive = RDP session. More than 3 unique hosts in 24h = suspicious
dc(ComputerName) > 3One account connecting to 3+ machines via RDP in 24h = lateral movement pattern
case() risk>10 hosts = possible worm/automated spread; >5 = investigate
SPL · PsExec / WMI Detection (EventID 7045) Sysmon + Security
index=* (sourcetype=WinEventLog:Security EventCode=7045) OR (sourcetype=XmlWinEventLog:Microsoft-Windows-Sysmon/Operational EventCode=1) earliest=-1h | eval is_psexec = if(match(ServiceFileName,"(?i)(psexec|PSEXESVC|paexec)"),"YES","NO") | eval is_wmi = if(match(CommandLine,"(?i)(wmic|wmiprvse)"),"YES","NO") | where is_psexec="YES" OR is_wmi="YES" | stats count, values(ComputerName) as targets by Account_Name, is_psexec
🔍 Phase 1 — Map the Movement ▾
Build complete timeline of host-to-host movements
Identify Patient Zero — which host was first compromised?
Was the Domain Controller accessed? → Emergency escalation
🌳 Lateral Movement Scope
Was the Domain Controller (DC) accessed?
✓ Yes → CRITICAL: Assume full domain compromise. Activate IR Plan.
✗ No → Continue
Were 5+ unique hosts accessed within 1 hour?
✓ Yes → HIGH: Possible automated spread. Contain all affected hosts.
✗ No → MEDIUM: Investigate source and intent
🚨 Phase 2 — Containment ▾
Isolate ALL affected hosts from the network
Disable the compromised account immediately
Force-close active RDP sessions on all affected hosts
Check for credential dumping — lsass.exe access? (Sysmon EventID 10)
