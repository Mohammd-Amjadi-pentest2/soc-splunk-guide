💾 What is Data Exfiltration?

The final stage of many attacks — the attacker transfers stolen data outside the network. Methods: large HTTPS uploads to cloud storage, FTP to attacker server, DNS tunneling, email attachments.

Detection requires a baseline: know what "normal" outbound traffic looks like for each host.

🔴 Key Detection Signals

1. Outbound volume spike — significantly above rolling average
2. Upload to unknown or unapproved services
3. After-hours activity from workstation accounts
4. Archive creation followed by upload (zip/rar then POST)
5. Large HTTP POST body to external IP

⚠
Context matters: 500MB upload to SharePoint = normal. 500MB to mega.nz at 11PM = critical alert.
SPL · Outbound Volume Anomaly Firewall / Palo Alto · -1h
index=* sourcetype="firewall" OR sourcetype="paloalto" action="allowed" direction="outbound" earliest=-1h | stats sum(bytes_out) as total_bytes, dc(dest_ip) as uniq_dest by src_ip | eval mb_out = round(total_bytes/1048576,2) | where mb_out > 100 | eval risk = case(mb_out>1000,"CRITICAL >1GB",mb_out>500,"HIGH >500MB",1=1,"MEDIUM") | sort -mb_out
SPL · Upload to Unapproved Cloud Services Proxy Logs · -24h
index=* sourcetype="proxy" (method="POST" OR method="PUT") earliest=-24h | where NOT match(dest_host,"(?i)(microsoft|office365|google|zoom|teams)") | eval upload_mb = round(cs_bytes/1048576,2) | stats sum(upload_mb) as total_mb, values(dest_host) as destinations by src_ip, user | where total_mb > 50 | sort -total_mb
🔍 Phase 1 — Scope Assessment ▾
Calculate exact data volume transferred — above 1GB is critical
Identify destination: dedicated IP or cloud service? What country?
Was transfer after business hours?
What file types were transferred (proxy logs)?
🌳 Exfiltration Decision Tree
Is destination in the organization's approved service list?
✓ Yes → Possible FP — verify with account owner
✗ No → Continue
Is volume >100MB to an unknown external IP?
✓ Yes → TRUE POSITIVE — Block + Escalate + Notify Legal/HR
✗ No → Investigate with Proxy/DLP logs
🚨 Immediate Response ▾
Block host network traffic — keep host alive for forensics
Block destination IP/domain at Firewall
Notify HR and Management — potential Insider Threat
Take forensic image before any changes
