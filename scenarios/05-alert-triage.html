⚡ What is Alert Triage?

SOC analysts receive thousands of alerts daily. Triage = prioritize + decide. Outcomes:

• True Positive — real attack → act
• False Positive — benign → tune the rule
• False Negative — missed attack → worst case
Goal: reduce MTTD (Mean Time to Detect) and prevent analyst fatigue.

📊 Splunk ES Notable Events

Correlation Searches fire Notable Events stored in index=notable. Each event has:
Risk Score (0–100) · Urgency (info/low/medium/high/critical) · Status · Owner

⚠
Critical Rule: Use Response Notes — NOT Audit Notes — to document analysis. Response Notes cannot be deleted and create a permanent audit trail.
SPL · Notable Events Prioritization index=notable · -24h
index=notable earliest=-24h | stats count as alert_count, dc(src_user) as unique_users, values(rule_name) as rules by urgency, status | eval priority = case(urgency="critical",1,urgency="high",2, urgency="medium",3,1=1,4) | sort priority
SPL · High-Volume Rules (FP Tuning Candidates) index=notable · -7d
index=notable earliest=-7d | stats count as fire_count, dc(src_ip) as uniq_src by rule_name | where fire_count > 100 | eval tuning = if(fire_count>500,"YES - HIGH PRIORITY","REVIEW") | sort -fire_count
⚡ Phase 1 — Alert Receipt ▾
Open Analyst Queue — address Critical and High first
Assign Notable Event to yourself — prevent duplicate work
Set Status to 'In Progress'
🌳 TP / FP / Tuning Decision
Does the activity match user/system context? (pentest, maintenance window)
✓ Yes → FP — document and suppress rule for this asset
✗ No → Continue
Has rule fired >50 times in 7 days with zero confirmed TPs?
✓ Yes → FP needing tuning — open Rule Optimization ticket
✗ No → Continue
Is alert confirmed by correlated evidence from multiple sources?
✓ Yes → TRUE POSITIVE — Begin Incident Response
✗ No → UNDETERMINED — Escalate to L2 with documentation
📝 Phase 2 — Documentation ▾
Write Response Notes — permanent, cannot be deleted, full audit trail
Document: what investigated, what found, decision and rationale
Set status to Resolved or Escalated with written justification
